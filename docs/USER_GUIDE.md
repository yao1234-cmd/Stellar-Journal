# 星迹使用指南 🌟

## 欢迎来到你的专属宇宙

星迹是一个将你的情感转化为可视化宇宙的应用。每一条记录，都会在你的星球上留下独特的痕迹。

## 🎮 基础操作

### 1. 查看星球

启动应用后，你会看到一个浮动在宇宙中的星球。

**交互方式：**
- 🖱️ **鼠标左键拖动**: 旋转星球
- 🖱️ **鼠标滚轮**: 放大缩小
- 👆 **触摸屏**: 单指滑动旋转，双指缩放

### 2. 创建记录

底部有一个记录面板，包含三种记录类型：

#### 💗 心情记录
- **作用**: 记录当前的情绪感受
- **效果**: 影响星球的大气层颜色
- **示例**: "今天见到了久违的朋友，心情特别好"

**颜色含义：**
- 🟡 明亮黄橙色 = 兴奋、快乐
- 🟢 薄荷绿蓝色 = 平静、满足
- 🔴 深红紫色 = 焦虑、愤怒
- 🔵 灰蓝色 = 悲伤、低落

#### ✨ 灵感记录
- **作用**: 捕捉突然闪现的想法
- **效果**: 在星球周围生成一颗闪亮的星星
- **示例**: "可以做一个自动整理笔记的工具"

**星星特性：**
- 每颗星星代表一个灵感
- 环绕星球缓慢公转
- 悬停可查看关键词

#### 🌳 思考记录
- **作用**: 记录深度思考和感悟
- **效果**: 在星球表面长出一棵树
- **示例**: "关于人际关系的思考：真诚比技巧更重要"

**树木特性：**
- 相同主题的思考会聚集成一棵树
- 思考越多，树木越茂盛
- 悬停显示主题和思考数量

### 3. 操作流程

1. **选择类型**: 点击心情/灵感/思考标签
2. **输入内容**: 在文本框中输入你的记录
3. **提交**:
   - 点击 📤 发送按钮
   - 或按 `Enter` 键（换行用 `Shift + Enter`）
4. **观察变化**: 星球会立即更新！

### 4. 查看历史

右侧有一个时光轴面板：

1. 点击 📅 **时光轴** 展开
2. 选择时间范围：7天 / 14天 / 30天
3. 查看历史颜色变化
4. 点击任意日期，星球会切换到那天的状态

**颜色块含义：**
- 每个颜色块代表某一天的情绪
- 颜色越深，情绪越强烈
- 灰色表示当天无记录

### 5. 查看统计

顶部右侧显示你的统计信息：
- 💗 心情记录数
- ✨ 灵感记录数
- 🌳 思考记录数
- 📅 连续活跃天数

## 💡 使用技巧

### 养成记录习惯

**每日三记录：**
1. **早上**: 记录今天的心情和计划
2. **白天**: 随时记录灵感闪现
3. **晚上**: 回顾并记录深度思考

### 情绪管理

**观察情绪模式：**
- 定期查看时光轴
- 注意颜色变化趋势
- 思考情绪波动的原因

**正念练习：**
- 记录前先深呼吸
- 诚实面对自己的情绪
- 不评判，只观察和记录

### 灵感管理

**快速捕捉：**
- 灵感稍纵即逝，立即记录
- 不必完整，关键词即可
- 后续可以展开为思考

### 思考沉淀

**主题归类：**
- 同类思考会自动聚类
- 观察哪些主题最常出现
- 形成个人知识体系

## 🎨 个性化你的星球

### 理解颜色语言

你的星球颜色由AI分析你的情绪后自动生成：

- **暖色调** (黄、橙、红)
  - 高能量、积极情绪
  - 兴奋、快乐、充满活力
  
- **冷色调** (蓝、紫、绿)
  - 平静、沉思
  - 可能悲伤或放松

- **饱和度**
  - 高饱和 = 情绪强烈
  - 低饱和 = 情绪平和

### 建造你的星球生态

**星星集群：**
- 多记录灵感会形成美丽的星环
- 不同类型灵感可能有不同颜色

**森林生长：**
- 持续记录同主题思考
- 观察思考之树的成长
- 形成独特的星球地貌

## ❓ 常见问题

### Q: 我的记录会被别人看到吗？
**A**: 绝对不会！你的所有记录都是私密的，仅你可见。我们承诺不会将你的数据用于任何第三方用途。

### Q: AI 如何分析我的情绪？
**A**: 使用先进的 GPT-4o 模型分析文本的情感倾向，包括积极/消极程度和激动/平静程度，然后映射为颜色。

### Q: 可以删除记录吗？
**A**: 当前版本暂不支持删除，但功能正在开发中。

### Q: 星球可以重置吗？
**A**: 暂不支持重置，每个记录都是你的回忆。

### Q: 支持导出数据吗？
**A**: 功能开发中，未来会支持导出为 JSON 或 PDF 格式。

### Q: 可以有多个星球吗？
**A**: 当前版本每个用户一个星球，未来可能支持多星球系统。
